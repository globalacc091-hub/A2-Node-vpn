# A2 Node Plan

## Цель

Создать премиальный VPN-продукт с темным интерфейсом, современным SPA-подходом, управлением WireGuard и платёжной интеграцией.

## Архитектура

- `frontend/` — визуальный интерфейс, реализация стилей и интерактивных анимаций
- `backend/` — Express API для состояния подключения, метрик, биллинга и раздачи фронтенда
- `vpn-core/` — генерация WireGuard-конфигураций и вспомогательная логика
- `billing/` — Stripe checkout и описание тарифов

## Текущий план

1. Убедиться, что `frontend/index.html` работает и реалистично показывает состояние подключения.
2. Развернуть `backend/index.js`, который будет служить статикой и предоставлять REST API.
3. Соединить `frontend` с backend-API для симуляции подключения и метрик.
4. Добавить `billing` с тарифами и checkout flow на Stripe.
5. Документировать запуск и структуру.

## Следующие шаги

- Подключить `frontend` к backend-эндпоинтам `/api/status`, `/api/connect`, `/api/disconnect`, `/api/metrics`.
- Внедрить `billing/stripe.js` с поддержкой `STRIPE_SECRET_KEY`.
- Добавить в документацию рекомендации по развёртыванию и использованию.
